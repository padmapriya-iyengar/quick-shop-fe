Quick Shop is a demo application to visualize and analyze the traffic & inventory data of a grocery store!!

  • Drag entities from the side pane to visualize and analyze data.
  • Individual cards dragged can be rearranged or deleted.
  • Dashboard can be configured with multiple charts.
  • Drag Order Summary onto the dashboard to view and analyse all the orders.
  • Drag Customers onto the dashboard to view and analyse all the customers.
  • Drag Products onto the dashboard to view and analyse all the products.
  • Drag Charts onto the dashboard to visualize data from all the entities.
  • Each chart has its own settings to change:
    • Chart Type - Bar/Pie/Doughnut
    • Data Source - Orders/Products/Customers
    • Populate By - Attribute for X Axis
    • Chart Title
    • X Axis Title
    • Y Axis Title
